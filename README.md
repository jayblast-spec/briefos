## ✨ Features

- 🌐 **URL ingestion** — paste any article or report URL and get a structured brief instantly
- 📝 **Text input** — paste raw text directly — research notes, email threads, documents
- 🗂️ **Structured output** — Summary, Key Points, Implications, Action Items, Source Quality
- 🎯 **Intelligence framing** — output designed for decision-making, not just summarization
- 🔍 **Source credibility signal** — flags content quality and bias indicators
- 📥 **Export to PDF or markdown** — share or archive your brief in one click
- ⚡ **No account needed** — input in, intelligence out, immediately

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/briefos.git
cd briefos
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
